# Build Guide

## Requirements

- Visual Studio Code
- PlatformIO IDE extension
- LILYGO T-Embed-CC1101
- USB data cable

## Steps

### 1. Open the project

Open the project folder in VS Code.

Do not open only `main.cpp`. Open the entire PlatformIO project folder.

### 2. Install dependencies

PlatformIO will install dependencies automatically according to `platformio.ini`.

### 3. Build

```bash
pio run
